🎮 Calculadora de Partidas Rankeadas
Projeto desenvolvido como parte do desafio prático da plataforma DIO. O objetivo é criar um script que calcula o saldo de partidas rankeadas de um jogador e determina seu nível (rank) com base na quantidade de vitórias.

🚀 Tecnologias e Conceitos Utilizados
Para a construção deste projeto, foram aplicados conceitos essenciais de lógica de programação:

Variáveis para armazenamento de dados.

Operadores aritméticos e de comparação.

Estruturas de Decisão (if, else if) para classificar os níveis do herói.

Funções para encapsular a lógica de cálculo e retorno.

🎯 Objetivo do Desafio
Criar uma função que receba como parâmetro a quantidade de vitórias e derrotas de um jogador.

O saldo de rankeadas deve ser calculado por: vitórias - derrotas.

O nível do jogador é determinado pela quantidade de vitórias, seguindo a regra abaixo:

Menor que 10: Ferro

Entre 11 e 20: Bronze

Entre 21 e 50: Prata

Entre 51 e 80: Ouro

Entre 81 e 90: Diamante

Entre 91 e 100: Lendário

Maior ou igual a 101: Imortal

📢 Saída Esperada
O programa deve exibir uma mensagem formatada como:

"O Herói tem de saldo de {saldoVitorias} está no nível de {nivel}"
